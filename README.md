# Chart.js
 
## Bar
Bar charts are great for comparing values across categories

## Line
Line charts are perfect for showing trends over time

## Area
An area chart is essentially a line chart with the space under the line filled in, making it better for showing cumulative values or emphasizing magnitude, while a line chart is clearer for comparing trends without visual emphasis on volume.

 
## Doughnut
It’s similar to a pie chart but with a hole in the middle

## Pie
It’s very similar to a doughnut chart, but without the hole in the middle


## Radar
Radar charts are great for comparing multiple variables across categories (like performance metrics, skill levels, etc.)
 

## Scatter Chart 
Scatter plots are useful for showing relationships between two variables 

## Bubble
Bubble charts are similar to scatter plots, but each point also has a size (radius) dimension, making them great for visualizing three variables at once (X, Y, and size)
